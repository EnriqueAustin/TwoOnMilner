# Ashby Manor Guest House

A React/Vite web application for Ashby Manor Guest House — a 4-star heritage guesthouse in Fresnaye, Cape Town.

## Getting Started

**Prerequisites:** Node.js 18+

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

## Available Scripts

- `npm run dev` — Start local development server
- `npm run build` — Build for production
- `npm run preview` — Preview the production build
- `npm run lint` — Lint the source files

## Project Structure

```
src/
  pages/       — Page components (Home, Rooms, Gallery, etc.)
  components/  — Shared and layout components
  lib/         — Utilities and constants
public/
  images/      — Local image assets
```
